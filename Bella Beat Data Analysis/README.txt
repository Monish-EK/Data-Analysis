Enhancing Wellness Insights: Bella Beat Data Analysis

This project involved performing a comprehensive data analysis on user interactions with Bella Beat’s wellness products. By analyzing data collected from activity trackers and health apps, the goal was to uncover actionable insights into user behavior and wellness trends. The project encompassed data collection and preprocessing, trend analysis, user segmentation, and predictive analytics. Key findings were visualized through intuitive dashboards and reports, providing a clear understanding of user habits and preferences. The insights gained were aimed at optimizing product features and improving user experience by tailoring recommendations based on data-driven evidence.

Skills: Data Analysis · Data Visualization · Data Cleaning · Tableau · SQL
